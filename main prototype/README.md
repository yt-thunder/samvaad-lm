# Samvaad LM Chatbot

## About the Project

This project is a simple, file-based chatbot designed to answer questions based on a predefined set of documents. It includes an admin dashboard for managing the chatbot and a feature for embedding the chatbot into other websites.

## Features

*   **Simple Chat Interface:** A clean and intuitive chat interface for interacting with the chatbot.
*   **Admin Dashboard:** A password-protected admin dashboard for managing the chatbot.
*   **Chatbot Testing:** A dedicated page for testing the chatbot's responses.
*   **Embeddable Chatbot:** A feature that allows you to embed the chatbot into any website using an iframe.
*   **Predefined Responses:** The chatbot uses a predefined set of questions and answers based on a `qna.txt` file.

## Getting Started

To get started with this project, simply open the `login.html` file in your web browser.

### Prerequisites

There are no special prerequisites for running this project. All you need is a modern web browser.

### Installation

No installation is required. Simply clone or download the project files.

## File Structure

*   `chatbot.html`: The main chat interface.
*   `script.js`: The JavaScript code for the chatbot interface.
*   `login.html`: The admin login page.
*   `login.js`: The JavaScript code for the login page.
*   `admin_dashboard.html`: The admin dashboard page.
*   `admin_dashboard.js`: The JavaScript code for the admin dashboard.
*   `embedded_chatbot.html`: The page used for embedding the chatbot.
*   `qna.txt`: A file containing the questions and answers for the chatbot.
*   `pictures/`: A folder containing images used in the application.
