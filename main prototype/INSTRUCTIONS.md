# Instructions

This document provides detailed instructions on how to use the Samvaad LM Chatbot application.

## Logging In

1.  Open the `login.html` file in your web browser.
2.  Enter the following credentials:
    *   **Username:** `admin`
    *   **Password:** `password`
3.  Click the "Login" button.

Upon successful login, you will be redirected to the admin dashboard.

## Admin Dashboard

The admin dashboard provides access to the following features:

*   **Test Your Chatbot:** This section allows you to test the chatbot's responses in a dedicated chat interface.
*   **Embed this chatbot into your college site:** This section provides a tool for embedding the chatbot into another website.

## Testing the Chatbot

1.  From the admin dashboard, click the "Test Your Chat Bot" button.
2.  This will open the `chatbot.html` page, where you can interact with the chatbot and test its responses.

## Embedding the Chatbot

1.  From the admin dashboard, go to the "Embed this chatbot into your college site" section.
2.  Enter the full URL of the website where you want to embed the chatbot in the "The college website URL" input field.
3.  Click the "Go Go Go" button.
4.  This will open a new page with the specified website and the chatbot displayed in the bottom right corner.
